# Models

This directory is reserved for model checkpoint files used in this project.

Model checkpoint files are not included in this repository due to file size.

## Expected Model File

The inference script expects a trained PatchCore checkpoint file.

Default expected structure:

```text
models/
└── model.ckpt
```

## How to Use

After training the PatchCore model or preparing a trained checkpoint, place the checkpoint file in this directory.

Then run inference using the default checkpoint path:

```bash
python inference/inference.py
```

By default, the inference script looks for:

```text
models/model.ckpt
```

You can also pass a custom checkpoint path using the `--ckpt` argument:

```bash
python inference/inference.py --ckpt path/to/model.ckpt
```

Example:

```bash
python inference/inference.py --ckpt models/model.ckpt --input data/test_images --threshold 0.5
```

## Checkpoint Source

During development, the checkpoint was generated by Anomalib after training PatchCore on the MVTec AD Cable dataset.

Training outputs generated by Anomalib may include checkpoints, logs, and result folders. These files are not committed to GitHub because they can be large and environment-specific.

## Notes

The checkpoint file is not committed to GitHub because model files can be large.

The MVTec AD dataset is also not included in this repository. Please download it from the official MVTec AD website.

This repository provides the training and inference workflow, while the dataset and checkpoint files should be prepared locally by the user.
